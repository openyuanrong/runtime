Metrics are a set of time-series data used to monitor the health and performance of openYuanrong systems and applications, so that problems can be identified and resolved promptly. The metric types currently supported by openYuanrong are as follows:

- [System Metrics](system-metrics.md): Include status information of clusters, nodes, and openYuanrong components.
- [Alarm Metrics](alarm-metrics.md): Include exception information for systems and dependent services.

openYuanrong supports multiple metrics export methods to integrate with your current operations platform. For details, see [Configure and Obtain Metrics](export-metrics.md).
