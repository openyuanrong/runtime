.. _invoke_g:

yr.Group.invoke
------------------------------------------------------------

.. py:method:: Group.invoke() -> None

    按照命运共享（fate-sharing）原则执行一组实例的创建。

    该函数用于创建一组具有相同命运的实例。组内的所有实例将被同时创建，
    如果其中任何一个实例创建失败，整个实例组都会被回滚。

    实例组的配置和相关选项由 `GroupOptions` 定义。
    返回：
        None。
