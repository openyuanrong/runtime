.. _suspend:

yr.Group.suspend
------------------------------------------------------------

.. py:method:: Group.suspend() -> None

    挂起（暂停）由 Group 管理的所有函数实例。

    该方法会在终止所有由 Group 创建和管理的函数实例之前，
    先对其实例状态和相关元数据进行 checkpoint（检查点）保存。
    这样在实例被杀死并释放资源后，仍然可以保留必要的实例信息。

    保存下来的 checkpoint 可在之后调用 `resume` 时使用，
    用于恢复并重新拉起这些函数实例。
    返回：
        None。
