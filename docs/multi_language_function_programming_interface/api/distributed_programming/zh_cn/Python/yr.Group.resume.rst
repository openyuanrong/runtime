.. _resume:

yr.Group.resume
------------------------------------------------------------

.. py:method:: Group.resume() -> None

    恢复由 Group 管理的所有函数实例。

    该方法会读取在 `suspend` 过程中保存的实例 checkpoint 信息，
    并基于这些信息重新创建和启动函数实例。

    实例在恢复后会使用已恢复的状态继续执行，
    而不是从全新的初始状态开始。
    返回：
        None。
